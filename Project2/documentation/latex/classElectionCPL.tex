\doxysubsection{Detailed Description}
\mbox{\hyperlink{classElectionCPL}{Election\+CPL}} class handles Closed \mbox{\hyperlink{classParty}{Party}} List (CPL) election. 

Responsible for reading in election data from a file in the same directory as src, reads election data from file, processes ballots, determines winners, and outputs results. Extends \mbox{\hyperlink{classElection}{Election}} class.
